Apa itu Tag? Tag adalah sebuah penanda atau kode yang dibuat dengan kurung patah ( <...> ), dan didalamnya terdapat namaTag-nya dan terdapat juga atributnya (pake jika dibutuhkan). Nah, Tag itu sendiri ada Tag pembuka dan Tag penutup (<...>namaTag</...>).

  • Heading

  • Image
  • Table
  • Form

Penggunaan NamaTag

Tag <p>

Digunakan untuk membuat sebuah isi / konten menjadi sebuah paragraf.

Kodenya:

<p>
	Lorem, ipsum dolor sit, amet consectetur adipisicing elit. Excepturi saepe ipsa soluta laudantium. Provident ab qui repudiandae tempora est rerum?
</p>

Outputnya:

Ini contoh Paragraf

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Suscipit amet similique quaerat sequi fugit debitis tenetur, blanditiis atque? Itaque molestiae vitae, enim nam amet delectus, consequatur repellendus aperiam dolores provident aliquam fugit quis! Quaerat ea esse placeat fugiat! Laborum totam maiores cum, quae odio delectus culpa consectetur? Nisi tenetur assumenda voluptatem libero animi praesentium culpa quia quis ea facere suscipit veniam, id ab asperiores quibusdam tempore voluptate modi odit officiis.

Tag <h1>, <h2>, <h3>, <h4>, <h5>, dan <h6>

Digunakan untuk menentukan ukuran besar kecilnya terhadap teksnya dan tag ini biasanya digunakan untuk sebuah judul halaman, judul iklan atau lainnya lah. Semakin besar angka pada Tagnya, semakin kecil teksnya.

Kodenya:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Outputnya:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Tag <ol>, <ul>, dan <dl>

Ketiga tag diatas digunakan untuk membuat sebuah list. Nah, untuk ketiga tag list tersebut, <ol> dan <ul> mempunyai tag tambahan, yaitu tag <li>. Sedangkan, tag <dl> tambahan tag-nya yaitu <dt> dan <dd>.

<ol> (order list) digunakan untuk membuat list secara terurut.

Kodenya:

<ol>
     <li>Senen</li>
     <li>Selasa</li>
     <li>Rabu</li>
     <li>Kamis</li>
     <li>Jumat</li>
     <li>Sabtu</li>
     <li>Minggu</li>
</ol>

Outputnya:

  1. Senen
  2. Selasa
  3. Rabu
  4. Kamis
  5. Jumat
  6. Sabtu
  7. Minggu

<ul> (un-order list) digunakan untuk membuat list tak berurut.

Kodenya:

<ul>
    <li>Mie Goreng</li>
    <li>Ayam Goreng</li>
    <li>Nasi Goreng</li>
</ul>

Outputnya:

  • Mie Goreng
  • Ayam Goreng
  • Nasi Goreng

<dl> (definision list) digunakan untuk mendefinisikan sebuah teks pada suatu list.

Kodenya:

 <dl>
     <dt>Ini Judul</dt>
     <dd>Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolorem, earum.</dd>
</dl>

Outputnya:

Ini Judul
Lorem ipsum dolor sit amet consectetur adipisicing elit. Ad, facilis?

Tag <a>

Tag anchor digunakan untuk merubah teks menjadi sebuah teks link, sebagai eksternal link maupun internal link. Tag anchor membutuhkan atribut tambahan yaitu href dan juga terdapat atribut target="_blank" yang fungsinya ketika link tersebut di klik, maka akan membuat tab baru di browser kalian.

Selengkapnya
Nah atribut target banyak nilainya, seperti:
1. _blank
2. _self
3. _top
4. _parent

Ada juga atribut download, digunakan sebagai link download.

Eksternal link ialah tujuannya ke url website didalam browser.

Kodenya:

 <a href="https://www.instagram.com">Klik disini</a>

Outputnya:

Klik disini

Internal Link ialah tujuannya ke url file lokal kita.

Kodenya:

<a href="index.html" target="_blank">Klik disini</a>

Outputnya:

Klik disini

Tag <img>

Tag image digunakan untuk menampilkan sebuah gambar/logo. Nah, tag image punya atribut khusus tambahan yaitu src="value" dan atribut lainnya seperti alt, title, width, height.
Ekstension pada gambar yaitu png, jpg, dan juga jpeg.

Selengkapnya

Jika kalian memakai gambar/logo yang ada di browser, besar kemungkinan akan terkena Copyright. Oleh karena itu, buat gambar/logo sendiri dengan software atau cari di browser gambar yang bebas pakai atau kunjungi website yang menyediakan gambar/logo yang bebas pakai.

Kodenya:

<img src="../img/gambar.png" alt="Tester" title="Komputer" width="250" height="200">

Outputnya:

Tag <table>

Didalam tag table ada tag tambahan lagi, yaitu <thead>, <tbody>, <tr> dan <th>. Terminologi pada tag table: 

   Field = Kolom
   Record = Baris

Tag table mempunyai atribut khusus juga, yaitu border, cellspacing, cellpading, colspan, dan rowspan.

Kodenya:

<table border="1" cellspacing="0" cellpadding="5">
     <thead>
         <tr>
            <th></th>
            <th>Kolom 1</th>
            <th>Kolom 2</th>
         </tr>
     </thead>
     <tbody>
         <tr>
            <th>Baris 1</th>
            <th>baris 1, kolom 1</th>
            <th>baris 1, kolom 2</th>
         </tr>
         <tr>
            <th>Baris 2</th>
            <th>baris 2, kolom 1</th>
            <th>baris 2, kolom 2</th>
          </tr>
     </tbody>
</table>

Outputnya:

Kolom 1 Kolom 2
Baris 1 baris 1, kolom 1 baris 1, kolom 2
Baris 2 baris 2, kolom 1 baris 2, kolom 2

Rowspan untuk membuat filed baru, sedangkan colspan untuk membuat record baru.

Kodenya:

<table border="1" cellspacing="0" cellpadding="5">
	<tr>
           <th rowspan="3">Rowspan 3</th>
	   <th colspan="2">Cosplan 2</th>
	</tr>
	<tr>
	   <th>2,1</th>
	   <th>2,2</th>
	</tr>
	<tr>
	   <th>3,1</th>
	   <th>3,2</th>
	</tr>
</table>

Outputnya:

Rowspan 3 Cosplan 2
2,1 2,2
3,1 3,2

Latihan:

Buatlah tabel dibawah ini. Ingat! gunakan rowspan dan colspan bersamaan dengan benar.

1,1 1,2 1,3 1,4
2,1 ? 2,4
3,1 3,4
4,1 4,2 4,3 4,4

Tag <form>

Didalam tag form terdapat tag tambahan, yaitu <input>, <label>, <select>, <button>, <textarea>. Nah didalam tag <select> ada tag tambahan lagi, yaitu <option>.

Tag <button> terdapat atribut type yang nilainya: 

   *- submit
   *- reset
   *- button

Selengkapnya
Tag <input> banyak sekali nilai pada atribut type, seperti:
*- text, untuk menuliskan sebuah teks
*- email, untuk menuliskan sebuah email
*- password, untuk menuliskan sebuah password
*- radio, hanya dapat memilih satu dengan tambahan atribut name
*- checkbox, bisa memilih lebih dari satu
*- date, menentukan tanggal lahir
*- time, menentukan waktu
*- datetime-local, menentukan tanggal lahir dan waktu
*- number, menentukan angka yang akan dipilih
*- month, menentukan tahun dan bulan
*- week, menentukan hari kosong atau jadwal
*- color, membuat button untuk mencari warna
*- file, membuat button untuk menaruh sebuah file
*- range, membuat bar persentasi
*- search, untuk menuliskan sesuatu yang ingin dicari
Ada juga atribut accept untuk menentukan file apa yang ingin diterima, nilainya seperti:
*- image/*
*- audio/*
*- video/*
Atribut action ialah:
Untuk memberikan tujuan ketika selesai membuat form-nya
Atribut method, nilainya:
*- get
*- post
Atribut autocomplete ialah:
Untuk memperlihatkan histori penulisan yang sebelumnya diketik, nilainya:
*- on
*- off
Atribut autofocus ialah:
Untuk memfokuskan titik cursor mouse-nya

Kodenya:

<form>
    <label for="username">username :</label>
    <input type="text" id="username">
    <br>
    <label for="password">password :</label>
    <input type="password" id="password">
    <br>
    <input type="radio" id="pria" name="jeniskelamin"><label for="pria">pria</label>
    <input type="radio" id="wanita" name="jeniskelamin"><label for="wanita">wanita</label>
    <br>
    <input type="checkbox" id="berenang"><label for="berenang">berenang</label>
    <input type="checkbox" id="menulis"><label for="menulis">menulis</label>
    <input type="checkbox" id="ngoding"><label for="ngoding">ngoding</label>
    <br>
    <textarea placeholder="COMMENT..."></textarea>
    <br>
    <select>
	<option>Jakarta</option>
	<option>Bandung</option>
	<option>Tangerang</option>
    </select>
    <br>
    <button type="submit">Kirim!</button>
</form>

Outputnya:







Latihan:

Buatlah form dibawah ini dengan bantuan <table>.

Tag Pendukung

Tag <b>, <i>, dan <u>

Tag bold digunakan untuk menebalkan teks, tag italic digunakan untuk memiringkan teks, dan tag underline digunakan untuk memberi garis bawah atau disebut juga garis penanda pada teks.

Kodenya:

<b>Selamat datang</b> <i>di website</i> <u>BangNomad</u>

Outputnya:

Selamat datang di website BangNomad

Tag <meter> dan <progress>

Kedua tag tersebut digunakan untuk membuat sebuah bar persentasi dan mempunyai atribut khusus untuk namaTag ini, yaitu min, max, dan value.

Kodenya:

<p>
    HTML <meter min="0" max="10" value="7"></meter>
    CSS <meter min="0" max="10" value="9"></meter>
    Javascript <meter min="0" max="10" value="4"></meter>
    Downloading : <progress min="0" max="100" value="69"></progress>
</p>

Outputnya:

HTML CSS Javascript

Downloading :

Tag <br> dan <hr>

Tag <br> dan <hr> digunakan sebagai tag pendukung di HTML. <br> untuk membuat baris baru dibawah paragraf, sedangkan <hr> untuk membuat sebuah garis horizontal.

Kodenya:

<p>
    Lorem ipsum dolor sit amet consectetur adipisicing elit. <br><br>Ratione perferendis impedit at modi dicta repellendus nulla cupiditate commodi debitis ducimus!<hr> Lorem ipsum dolor sit amet consectetur adipisicing elit. Deserunt, illum reiciendis!
</p>

Outputnya:

Lorem ipsum dolor sit amet consectetur adipisicing elit.

Ratione perferendis impedit at modi dicta repellendus nulla cupiditate commodi debitis ducimus!

Lorem ipsum dolor sit amet consectetur adipisicing elit. Deserunt, illum reiciendis!

Tag <pre> dan <mark>

Tag <pre> untuk membuat spasi menjadi lebih panjang jarak antar kata nya. Karena kalo pake Tag <p> spasi sebanyak mungkin ga bakal ngaruh. Sedangkan tag <mark> untuk membuat label pada teks.

Kodenya:

<pre>Halo, nama saya Muhammad    <mark>Naufal</mark>    Thufailah.</pre>

Outputnya:

Halo, nama saya Muhammad    Naufal    Thufailah.

Tag <sup> dan <sub>

Untuk membuat teks menjadi berpangkat, dan tag ini untuk kalkulasi perhitungan seperti kalkulator.

Kodenya:

<p>2<sup>2</sup> + 2<sup>3</sup> = 12 <br> H<sub>2</sub>O adalah ...</p>

Outputnya:

22 + 23 = 12
H2O adalah ...

Tag <dialog>

Untuk membuat kotak dialog antar user dengan tambahan atribut open

Kodenya:

 <dialog open>Selamat Datang</dialog>

Outputnya:

Selamat Datang

Tag <code> dan <xmp>

Digunakan untuk menampilkan sebuah source code yang kita kodingin ke dalam website.

Kodenya:

<code>console.log('ok');</code>
          
<xmp>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Officia, voluptatibus.</xmp>

Outputnya <code>:

console.log('ok');

Outputnya <xmp>:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Fugiat, eligendi?

Tag di HTML 5

Tag <colgroup> dan <col>

Digunakan untuk bantuan pada tag <table> yang mana tugas dari <colgroup> dan <col> adalah meng-grup-kan isi dari tabel tersebut, sehingga tag <col> mempunyai atribut tambahan, yaitu span.

Kodenya:

<table border="1">
     <caption>Tabel Tester</caption>
     <colgroup>
	<col span="1" style="background-color: #0ff;">
	<col span="2" style="background-color: #f0f;">
	<col span="1" style="background-color: #ff0;">
     </colgroup>
     <tr>
	<th>ID</th>
	<th>Nama</th>
	<th>Umur</th>
	<th>Tempat</th>
     </tr>
     <tr>
	<th>1</th>
	<th>Kasep</th>
	<th>22</th>
	<th>Bandung</th>
     </tr>
     <tr>
	<th>2</th>
	<th>Ismet</th>
	<th>19</th>
	<th>Tangerang</th>
     </tr>
     <tr>
	<th>3</th>
	<th>Romli</th>
	<th>28</th>
	<th>Jogja</th>
     </tr>
</table>

Outputnya:

Tabel Tester
ID Nama Umur Tempat
1 Kasep 22 Bandung
2 Ismet 19 Tangerang
3 Romli 28 Semarang

Tag <details> dan <datalist>

Tag <details> mempunyai tag tambahan, yaitu <summary> ialah untuk membuat paragraf yang di hide dan ketika di klik akan menampilkan data detailnya. Sedangkan tag <datalist> tag tambahannya <option> ialah untuk membuat list data dengan cara menuliskannya.

Kodenya:

<details>
     <summary>Details</summary>
     <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Reiciendis, laudantium! Atque, blanditiis, at expedita repudiandae inventore cumque suscipit provident rerum mollitia sint iusto nisi sapiente.</p>
</details>
<br><br>
<input list="kota">
<datalist id="kota">
    <option value="Jakarta"></option>
    <option value="Bekasi"></option>
    <option value="Cirebon"></option>
</datalist>
<input type="submit">

Outputnya:

Contoh <details>

Details

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Reiciendis, laudantium! Atque, blanditiis, at expedita repudiandae inventore cumque suscipit provident rerum mollitia sint iusto nisi sapiente.

Personal Info First Name :

<audio controls>
     <source src="audio.mp3" type="audio/mp3">

Tag <video> dan <embed>

Keduanya digunakan untuk menampilkan sebuah video. Tag ini mempunyai atribut seperti autoplay, controls, loop, muted, preload, src, poster, width dan height.

Selengkapnya

Atribut autoplay untuk memulai video saat pertama kali membuka web
Atribut controls untuk menampilkan kontrol video
Atribut loop untuk mengulang video tanpa henti
Atribut muted untuk memuted suara video
Atribut poster untuk merubah thumbnail video menjadi objek lainnya

Kodenya:

<video controls width="500px" height="500px" loop>
     <source src="video.mp4" type="video/mp4">
</video>

<embed src="video.mp4" type="video/mp4" width="250px" height="200px">

Outputnya:
