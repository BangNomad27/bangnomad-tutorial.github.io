Apa itu Tag? Tag adalah sebuah penanda atau kode yang dibuat dengan kurung patah ( <...> ), dan didalamnya terdapat namaTag-nya dan terdapat juga atributnya (pake jika dibutuhkan). Nah, Tag itu sendiri ada Tag pembuka dan Tag penutup (<...>namaTag</...>).

Penggunaan NamaTag

  • Paragraf
  • Bold, Italic, Underline
  • Heading
  • List
  • Hyperlink
  • Image
  • Table
  • Form

Penggunaan NamaTag

Tag <p>

Digunakan untuk membuat sebuah isi / konten menjadi sebuah paragraf.

Kodenya:

<p> Lorem ipsum dolor sit amet consectetur adipisicing elit. Tempore possimus omnis officia, repellat harum consectetur rem magnam, quod odit suscipit voluptates quam veniam voluptatem deserunt ullam blanditiis quaerat sunt accusantium! </p>

Outputnya:

Ini contoh Paragraf

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Suscipit amet similique quaerat sequi fugit debitis tenetur, blanditiis atque? Itaque molestiae vitae, enim nam amet delectus, consequatur repellendus aperiam dolores provident aliquam fugit quis! Quaerat ea esse placeat fugiat! Laborum totam maiores cum, quae odio delectus culpa consectetur? Nisi tenetur assumenda voluptatem libero animi praesentium culpa quia quis ea facere suscipit veniam, id ab asperiores quibusdam tempore voluptate modi odit officiis.

Tag <b>, <i>, dan <u>

Tag bold digunakan untuk menebalkan teks, tag italic digunakan untuk memiringkan teks, dan tag underline digunakan untuk memberi garis bawah atau disebut juga garis penanda pada teks.

Kodenya:

<b>Selamat datang</b> <i>di website</i> <u>BangNomad</u>

Outputnya:

Selamat datang di website BangNomad

Tag <h1>, <h2>, <h3>, <h4>, <h5>, dan <h6>

Digunakan untuk menentukan ukuran besar kecilnya terhadap teksnya dan tag ini biasanya digunakan untuk sebuah judul halaman, judul iklan atau lainnya lah. Semakin besar angka pada Tagnya, semakin kecil teksnya.

Kodenya:

<h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3> <h4>Heading 4</h4> <h5>Heading 5</h5> <h6>Heading 6</h6>

Outputnya:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Tag <ol>, <ul>, dan <dl>

Ketiga tag diatas digunakan untuk membuat sebuah list. Nah, untuk ketiga tag list tersebut, <ol> dan <ul> mempunyai tag tambahan, yaitu tag <li>. Sedangkan, tag <dl> tambahan tag-nya yaitu <dt> dan <dd>.

<ol> (order list) digunakan untuk membuat list secara terurut.

Kodenya:

<ol> <li>Senen</li> <li>Selasa</li> <li>Rabu</li> <li>Kamis</li> <li>Jumat</li> <li>Sabtu</li> <li>Minggu</li> </ol>

Outputnya:

  1. Senen
  2. Selasa
  3. Rabu
  4. Kamis
  5. Jumat
  6. Sabtu
  7. Minggu

<ul> (un-order list) digunakan untuk membuat list tak berurut.

Kodenya:

<ul> <li>Mie Goreng</li> <li>Ayam Goreng</li> <li>Nasi Goreng</li> </ul>

Outputnya:

  • Mie Goreng
  • Ayam Goreng
  • Nasi Goreng

<dl> (definision list) digunakan untuk mendefinisikan sebuah teks pada suatu list.

Kodenya:

<dl> <dt>Ini Judul</dt> <dd>Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolorem, earum.</dd> </dl>

Outputnya:

Ini Judul
Lorem ipsum dolor sit amet consectetur adipisicing elit. Ad, facilis?

Tag <a>

Tag anchor digunakan untuk merubah teks menjadi sebuah teks link, sebagai eksternal link maupun internal link. Tag anchor membutuhkan atribut tambahan yaitu href="value" dan juga terdapat atribut target="_blank" yang fungsinya ketika link di klik, maka akan membuat tab baru di browser kalian.

Eksternal link ialah tujuannya ke url website didalam browser.

Kodenya:

<a href="https://www.instagram.com">Klik disini</a>

Outputnya:

Klik disini

Internal Link ialah tujuannya ke url file lokal kita.

Kodenya:

<a href="index.html" target="_blank">Klik disini</a>

Outputnya:

Klik disini

Tag <img>

Tag image digunakan untuk menampilkan sebuah gambar/logo. Nah, tag image punya atribut khusus tambahan yaitu src="value" dan atribut lainnya seperti alt, title, width, height.
Ekstension pada gambar yaitu png, jpg, dan juga jpeg.

Jika kalian memakai gambar/logo yang ada di browser, besar kemungkinan akan terkena Copyright. Oleh karena itu, buat gambar/logo sendiri dengan software atau cari di browser gambar yang bebas pakai atau kunjungi website yang menyediakan gambar/logo yang bebas pakai.

Kodenya:

<img src="../img/gambar.png" alt="Tester" title="Komputer" width="200" height="200">

Outputnya:

Tag <table>

Didalam tag table ada tag tambahan lagi, yaitu <thead>, <tbody>, <tr> dan <th>. Terminologi pada tag table: 

   Field = Kolom
   Record = Baris

Tag table mempunyai atribut khusus juga, yaitu border, cellspacing, cellpading, colspan, dan rowspan.

Kodenya:

<table border="1" cellspacing="0" cellpadding="5"> <thead> <tr> <th></th> <th>Kolom 1</th> <th>Kolom 2</th> </tr> </thead> <tbody> <tr> <th>Baris 1</th> <th>baris 1, kolom 1</th> <th>baris 1, kolom 2</th> </tr> <tr> <th>Baris 2</th> <th>baris 2, kolom 1</th> <th>baris 2, kolom 2</th> </tr> </tbody> </table>

Outputnya:

Kolom 1 Kolom 2
Baris 1 baris 1, kolom 1 baris 1, kolom 2
Baris 2 baris 2, kolom 1 baris 2, kolom 2

Rowspan untuk membuat filed baru, sedangkan colspan untuk membuat record baru.

Kodenya:

<table border="1" cellspacing="0" cellpadding="5"> <tr> <th rowspan="3">Rowspan 3</th> <th colspan="2">Cosplan 2</th> </tr> <tr> <th>2,1</th> <th>2,2</th> </tr> <tr> <th>3,1</th> <th>3,2</th> </tr> </table>

Outputnya:

Rowspan 3 Cosplan 2
2,1 2,2
3,1 3,2

Latihan:

Buatlah tabel dibawah ini. Ingat! gunakan rowspan dan colspan bersamaan dengan benar.

1,1 1,2 1,3 1,4
2,1 ? 2,4
3,1 3,4
4,1 4,2 4,3 4,4

Tag <form>

Didalam tag form terdapat tag tambahan, yaitu <input>, <label>, <select>, <button>, <textarea>.
Nah didalam tag <select> ada tag tambahan lagi, yaitu <option>

Tag <input> terdapat atribut type yang nilainya: 

   *- text
   *- password
   *- radio
   *- checkbox

Tag <button> terdapat atribut type yang nilainya: 

   *- submit
   *- reset
   *- button

Kodenya:

<article> <form> <label for="username">username :</label> <input type="text" id="username"> <br> <label for="password">password :</label> <input type="password" id="password"> <br> <input type="radio" id="pria" name="jeniskelamin"><label for="pria">pria</label> <input type="radio" id="wanita" name="jeniskelamin"><label for="wanita">wanita</label> <br> <input type="checkbox" id="berenang"><label for="berenang">berenang</label> <input type="checkbox" id="menulis"><label for="menulis">menulis</label> <input type="checkbox" id="ngoding"><label for="ngoding">ngoding</label> <br> <textarea placeholder="COMMENT..."></textarea> <br> <select> <option>Jakarta</option> <option>Bandung</option> <option>Tangerang</option> </select> <br> <button type="submit">Kirim!</button> </form>

Outputnya:







Latihan:

Buatlah form dibawah ini dengan bantuan <table>.
