Property CSS 1

*- Font Size

Digunakan untuk menentukan ukuran font pada sebuah teks

Kodenya:

selector { font-size: 20px; }

Outputnya:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Vel, quisquam!

*- Font Family

Digunakan untuk menentukan jenis fontnya seperti yang di inginkannya

Kodenya:

selector { font-family: verdana; }

Outputnya:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Sequi, obcaecati.

*- Font Weight

Digunakan untuk menentukan ketebalan fontnya, bedanya dengan tag <b> ialah property ini bisa mengatur ketebalan fontnya. Nilainya: normal, bold, bolder dan 500-900

Kodenya:

selector { font-weight: 900; }

Outputnya:

Lorem ipsum, dolor sit amet consectetur adipisicing elit.

*- Font Style

Digunakan untuk menentukan kemiringan pada fontnya, property ini sama seperti tag <i> di HTML. Nilainya: normal, italic dan oblique

Kodenya:

selector { font-style: oblique; }

Outputnya:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Incidunt, reiciendis!

*- Font Variant

Digunakan untuk membuat huruf menjadi capital walaupun teks default ga capital. Nilainya: normal, small-caps, inherit dan unset

Kodenya:

selector { font-variant: small-caps; }

Outputnya:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Enim, distinctio.

Masih banyak lagi property font lainnya seperti: font-kerning, font-stretch, font-synthesis dan lain lain.

*- Text Align

Digunakan untuk menentukan kerataan fontnya. Nilainya: left, center, right dan justify

Kodenya:

selector { text-align: left; }
selector { text-align: center; }
selector { text-align: right; }

Outputnya:

Rata Kiri

Rata Tengah

Rata Kanan

*- Text Indent

Digunakan untuk memberi ruang kosong diawal paragraf seperti menggunakan simbol

&nbsp;

Kodenya:

selector { text-indent: 20px; }

Outputnya:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatum, odio qui quos incidunt fugiat nihil impedit, a dolor consectetur, provident mollitia. Odit quasi expedita in iste eius dolor ipsa quam.

*- Text Decoration

Digunakan untuk mendekorasi sebuah teks dengan menambahkan sebuah garis pada teksnya. Nilainya: none, underline, overline dan line-through

Kodenya:

selector { text-decoration: underline; }
selector { text-decoration: overline; }
selector { text-decoration: line-through; }

Outputnya:

Underline, Overline dan Line-Through

*- Text Transform

Digunakan untuk membuat gaya pada fontnya. Nilainya: none lowercase, uppercase, dan capitalize

Kodenya:

selector { text-transform: lowercase; }
selector { text-transform: uppercase; }
selector { text-transform: capitalize; }

Outputnya:

LOWERCASE = LOWERCASE, uppercase = uppercase dan capitalize = capitalize

Masih banyak lagi property lainnya seperti: text-anchor, text-shadow, text-rendering, text-overflow, text-orientation dan lain lain.

*- Background dan Color

Kedua property ini digunakan untuk memberikan warna, property background untuk mewarna halamannya sedangkan property color untuk mewarna teksnya. Tapi untuk property background saya ga tampilkan outputnya hanya kodenya aja biar bisa di copy dan dicoba

Selengkapnya
Beberapa property untuk background, yaitu:
*- background-color nilainya: nama warna | hexa-desimal | rgb() | rgba() | hsla()
*- background-image nilainya: url() | linear/radial-gradient
*- background-size nilainya: count manually | cover
*- background-repeat nilainya: no-repeat
*- background-position nilainya: count manually

Untuk property color sama seperti background-color nilainya:
*- color nilainya: nama warna | hexa-desimal | rgb() | rgba() | hsla()

Count Manually ialah hitungnya secara manual 0 0; angka pertama horizontal dan angka kedua vertikal. Selebihnya lihat di kodenya
