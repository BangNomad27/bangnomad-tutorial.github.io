*- Background dan Color

Kedua property ini digunakan untuk memberikan warna terhadap elemennya.

Selengkapnya
Beberapa property untuk background, yaitu:
*- background-color
Untuk memberikan warna pada halaman / background webnya. Nilainya: nama warna, hexa-desimal dan rgb()
*- background-image
Untuk memberikan sebuah gambar pada backgroundnya. Nilainya: url(), linear-gradient dan radial-gradient
*- background-size:
Untuk menentukan ukuran gambarnya. Nilainya: count manually dan cover
*- background-repeat
Untuk menentukan gambarnya berulang kali atau single image. Nilainya: no-repeat
*- background-position
Untuk menentukan posisi gambarnya. Nilainya: count manually, left, right dan center
Count manually adalah nilai yang diatur manual oleh programmernya, terdapat dua nilai, nilai pertama untuk menentukan nilai secara horizontal sedangkan nilai kedua untuk menentukan nilai secara vertikal. Dan masih ada beberapa property lainnya seperti: background-clip, background-attachment, background-blend-mode dan background-origin

Untuk property color sama seperti background-color nilainya:
*- color
Untuk memberikan warna pada teksnya. Nilainya: nama warna, hexa-desimal, rgb(), rgba() dan hsla()
Lalu cara pemanggilan warnanya, yaitu:
*- nama warna: red; | green; | blue;
*- hexa-desimal: #ff0000; | #008000; | #0000ff;
*- rgb(): rgb(255,0,0); | rgb(0,0,255); | rgb(0,128,0);

Kode property background:

selector { background-color: red; }
selector { background-image: url(image.png); }
selector { background-size: cover; }
selector { background-repeat: no-repeat; }
selector { background-position: 0 100px; }

Kode property color:

selector { color: red; }
selector { color: #008000; }
selector { color: rgb(0,128,0); }

*- Pseudo-Class

Pseudo-class sangat penting untuk CSS, pseudo-class juga bisa diartikan sebagai simbol pada selector. Berikut, beberapa simbol pada selectornya:

*- *

Simbol bintang (*) ini digunakan untuk mengubah keseluruhan elemen pada CSS dan hati hati ketika menggunakan simbol ini, karena bisa dibilang sebagai CSS Reset dan biasanya juga margin dan padding yang di reset

Selengkapnya

Kodenya:

* {
     margin: 0;
     padding: 0;
}

*- #x

Simbol pagar (#) digunakan untuk memberi style pada namaTag yang memiliki atribut id. Namun ketika ingin menggunakan simbol ini hati hati, karena nilai spesificity atribut id sangat besar

Selengkapnya

Kodenya:

#container {
     background-color: #f1f2f6;
     color: #111;
}

*- .x

Simbol titik (.) digunakan untuk memberi style pada namaTag yang memiliki atribut class

Selengkapnya

Kodenya:

.container {
     background-color: #333;
     color: #fff;
}

*- x y

Spasi digunakan menargetkan ke tujuan yang lebih spesifik

Selengkapnya

Kodenya:

     border: none;

Property CSS 2

*- Display

Display adalah sebuah display yang ada disetiap namaTag, seperti tag <div> secara default memiliki displaynya block dan kita bisa mengubah display tersebut dengan memanggil property display. Ada beberapa nilainya yang bisa kita ubah, seperti: none, inline, block dan inline-block

Selengkapnya
Penjelasan nilai pada property display:
*- none, untuk menghilangkan sebuah elemen
*- inline, tidak dapat menambahkan garis baru ketika dibuat (secara default)
*- block, dapat mengatur ukuran elemennya
*- inline-block dapat menambahkan garis baru serta mengatur ukurannya

Kodenya:

selector { display: inline-block; } /* atau nilai lainnya */

*- Overflow

Overflow adalah sebuah property yang digunakan untuk mengatur perilaku elemen yang kasusnya keluar dari parent-nya
Apa itu Parent? Parent adalah sebuah pembungkus/box yang didalamnya mempunyai beberapa tag-tag, dan tag disebut juga sebagai anak. Nilai pada property overflow seperti: visible, auto, hidden dan scroll

Selengkapnya
Penjelasan nilai pada property overflow:
*- visible, adalah nilai default disetiap property
*- auto, CSS akan otomatis menambahkan scroll ketika kontennnya ga cukup, tapi jika cukup maka scroll akan hilang
*- hidden, CSS akan menyembunyikan konten ketika melebihi parentnya
*- scroll, memunculkan sebuah scroll seperti nilai auto, bedanya ketika kontennya cukup scrollnya tetep ada

Kodenya:

selector { overflow: auto; } /* atau nilai lainnya */

*- Clear

Property clear bertugas membersihkan atau menghentikan property float. Karena kalo ga dihentikan, maka akan berantakan atau ga responsive parent-nya. Seperti yang kalian lihat output diatas, gambar tersebut keluar dari parent-nya

Selengkapnya
Terdapat tiga nilai pada property clear, yaitu:
*- left, digunakan ketika selector mempunyai float bernilai 'left'
*- right, digunakan ketika selector mempunyai float bernilai 'right'
*- both, digunakan ketika selector mempunyai float bernilai 'left' dan 'right'

Terdapat tiga cara menghentikan/membersihkan property float, yaitu:
*- Menggunakan property overflow bernilai 'auto'
*- Menggunakan <div> kosong
*- Menggunakan Teknik Micro Clearfix

Untuk ngakalin dengan micro clearfix, ada dibawah ini syntaxnya

/* Micro Clearfix */
.cf:before,
.cf:after {
     content: " ";
     display: table;
}
.cf:after {
     clear: both;
}
.cf {
     *zoom: 1;
}

*- Position

Position adalah property penting dan agak rumit untuk pemula pelajari. Position dapat melakukan tata letak dari elemen elemen pada HTML, sesuai dengan keinginan programmer. Nilai pada property position, yaitu: static, relative, absolute dan fixed

Selengkapnya
Penjelasan nilai pada property position:
1. static (nilai default):
*- Jika tidak memberikan property position pada elemen, maka static adalah nilai defaultnya
*- Jika menggunakan selain position static, disebut sebagai non-static. Maka, akan membuat elemen menjadi berbeda dimensi dari elemen lainnya
*- Elemen yang diberikan position non-static, dapat menggunakan property tambahan, yaitu: top, right, bottom dan left

Outputnya:

2. relative:
*- Ketika menggerakkan elemen dengan posisi relative, maka ruang yang ditempati oleh elemen tersebut tetep masih ada
*- Ketika menggerakkan elemen dengan posisi relative, maka elemen akan bergerak relative terhadap posisi semulanya
*- Jika memberikan property top: 0; dan left: 0;, maka elemen tersebut ga berubah dari posisi awalnya

Outputnya:

3. absolute:
*- Ketika menggerakkan elemen dengan posisi absolute, maka ruang yang ditempati oleh elemen tersebut dianggap tidak ada
*- Ketika menggerakkan elemen dengan posisi absolute, maka elemennya bergerak relatif terhadap posisi parent-nya
*- Jika memberi property top: 0 dan left: 0, maka elemen akan menempel pada parent-nya (parent defaultnya adalah <body>)

Outputnya:

4. fixed:
*- Ketika menggerakkan elemen tersebut menggunakan top, right, bottom, left, maka dia akan selalu bergerak relatif terhadap windows-nya si browser (windows si browser adalah <body>)
*- Ketika di scroll halamannya, maka elemen tersebut akan terkunci diposisi tertentu
Maaf, tidak ada contoh Output

Kodenya:

selector { position: relative; } /* atau lainnya */

*- Box Model

Box model adalah sebuah box yang didalamnya memiliki elemen elemen. Box model memiliki 4 property pendukungnya, seperti: margin, padding, border dan content. Untuk lebih jelasnya bisa kalian lihat melalui CTRL + SHIFT + I atau klik kanan dan pilih 'inspeksi

Selengkapnya
Pembahasan mengenai shorthand dari ke-4 property tersebut:
*- 1 Nilai, nilai untuk ke-4 sisi (semua sisi)
*- 2 Nilai, nilai pertama untuk horizontal dan nilai kedua untuk vertikal
*- 3 Nilai, nilai pertama untuk atas (top), nilai ke-2 untuk horizontal dan nilai ke-3 untuk bawah (bottom)
*- 4 Nilai, nilainya seperti arah jarum jam

Kodenya:

selector { margin: 25px auto; }
selector { padding: 10px 100px 10px; }
selector { border: 10px solid #333; }

*- Overlapping Margin, dua margin saling bertemu
*- Negatif Margin, dapat diberikan nilai negarif (-)
*- Auto, dapat diberikan nilai auto (nilainya menyesuaikan si elemen)

Kodenya:

selector { margin: 25px auto; }

*- Before & After

String:
digunakan untuk menuliskan sebuah teks apapun isinya dan bisa diberikan style, atau selain teks bisa diberikan Enitas Code yang biasa digunakan sebagai icon atau glyph
Image:
Dapat memberikan url() untuk menuju ke sebuah gambar, syntax-nya seperti ini: content: url(../img/image.png);, kekurangannya adalah gambar tersebut ga bisa di resize dan ga bisa diberikan gradient
None:
Dapat di kosongkan kontennya seperti ini: content: " ";
Counter:
Untuk menghitung jumlah elemen yang ada di HTML

kodenya:

selector::before { content: "Hi, "; }
selector::after { content: " World!"; }
