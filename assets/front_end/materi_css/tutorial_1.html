*- Background dan Color

Kedua property ini digunakan untuk memberikan warna terhadap elemennya.

Selengkapnya
Beberapa property untuk background, yaitu:
*- background-color
Untuk memberikan warna pada halaman / background webnya. Nilainya: nama warna, hexa-desimal dan rgb()
*- background-image
Untuk memberikan sebuah gambar pada backgroundnya. Nilainya: url(), linear-gradient dan radial-gradient
*- background-size:
Untuk menentukan ukuran gambarnya. Nilainya: count manually dan cover
*- background-repeat
Untuk menentukan gambarnya berulang kali atau single image. Nilainya: no-repeat
*- background-position
Untuk menentukan posisi gambarnya. Nilainya: count manually, left, right dan center
Count manually adalah nilai yang diatur manual oleh programmernya, terdapat dua nilai, nilai pertama untuk menentukan nilai secara horizontal sedangkan nilai kedua untuk menentukan nilai secara vertikal. Dan masih ada beberapa property lainnya seperti: background-clip, background-attachment, background-blend-mode dan background-origin

Untuk property color sama seperti background-color nilainya:
*- color
Untuk memberikan warna pada teksnya. Nilainya: nama warna, hexa-desimal, rgb(), rgba() dan hsla()
Lalu cara pemanggilan warnanya, yaitu:
*- nama warna: red; | green; | blue;
*- hexa-desimal: #ff0000; | #008000; | #0000ff;
*- rgb(): rgb(255,0,0); | rgb(0,0,255); | rgb(0,128,0);

Kode property background:

selector { background-color: red; }
selector { background-image: url(image.png); }
selector { background-size: cover; }
selector { background-repeat: no-repeat; }
selector { background-position: 0 100px; }

Kode property color:

selector { color: red; }
selector { color: #008000; }
selector { color: rgb(0,128,0); }

*- Pseudo-Class

Pseudo-class sangat penting untuk CSS, pseudo-class juga bisa diartikan sebagai simbol pada selector. Berikut, beberapa simbol pada selectornya:

*- *

Simbol bintang (*) ini digunakan untuk mengubah keseluruhan elemen pada CSS dan hati hati ketika menggunakan simbol ini, karena bisa dibilang sebagai CSS Reset dan biasanya juga margin dan padding yang di reset.

Selengkapnya

Kodenya:

* {
     margin: 0;
     padding: 0;
}

*- #x

Simbol pagar (#) ini digunakan untuk memberi style pada namaTag yang memiliki atribut id. Namun ketika ingin menggunakan simbol ini hati hati, karena nilai spesificity atribut id sangat besar.

Selengkapnya

Kodenya:

#container {
     background-color: #f1f2f6;
     color: #111;
}

*- .x

Simbol titik (.) ini digunakan untuk memberi style pada namaTag yang memiliki atribut class.

Selengkapnya

Kodenya:

.container {
     background-color: #333;
     color: #fff;
}
