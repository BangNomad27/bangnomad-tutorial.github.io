Digunakan untuk melakukan perubahan pada box model yang dimana sudut sudut box model tidak lancip, melainkan menjadi tumpul. Semakin besar nilainya, box model akan tampak menjadi bulat/lingkaran.

Property border-radius dan shorthand-nya:
*- border-top-left-radius
*- border-top-right-radius
*- border-bottom-left-radius
*- border-bottom-right-radius
*- border-radius

Kodenya:

selector { border-radius: 50%; }

*- Opacity

Opacity digunakan untuk membuat suatu objek menjadi transparan. Opacity nilainya dari 0 sampai 1, yang bearti 0 = 0% sedangkan 1 = 100%. Contoh: nilainya 0.5 = 50% atau contoh lain bisa juga tanpa 0: nilainya .25 = 0.25 = 25%

Selengkapnya

Pada contoh output dibawah ini, warna biru saya berikan opacity dengan nilai 1 = 100%, tapi tetap transparan. Itu terjadi karena warna biru didalam parent warna oren, gimana cara atasinya agar warna oren tetep transparan sedangkan warna biru ga transparan? disitulah kita butuh RGBa / HSLa

Kodenya:

selector { opacity: 0.7; }

Outputnya:

*- RGBa & HSLa

RGBa artinya Red Green Blue Alpha, RGBa digunakan untuk memberi transparansi juga seperti opacity, bedanya RGBa memberikan transparansi terhadap warna yang kita pilih/seleksi. Implementasinya sama seperti rgb()

HSLa artinya Hue Saturation Lightness Alpha, HSLa sama seperti RGBa. Bedanya terdapat satuan persen (%) pada HSLa dan HSLa akan lebih transparan dibanding RGBa

Selengkapnya

Perbedaan Opacity dengan RGBa/HSLa yaitu, opacity digunakan untuk mengtransparansikan elemen, jadi opacity ga peduli didalamnya ada elemen bewarna apa ia (opacity) tetep transparan. Sedangkan RGBa/HSLa digunakan untuk mengtransparansikan hanya diberikan kepada elemen yang dipilih, tetapi image ga bisa karena image ga punya background-color serta color-nya

Kodenya:

selector { background-color: rgba(255,102,0,0.5); } /* atau */ selector { background-color: hsla(24,100%,63%,0.5); }

*- Box-Shadow

Digunakan untuk memberikan sebuah bayangan terhadap box modelnya. Box-shadow terdapat 2 cara, yaitu: non-opsional dan opsional

Selengkapnya
Non-opsional, seperti:
*- x-offset
*- y-offset
*- blur
*- color
Opsional, seperti:
*- inset
*- spread

Penjelasan non-opsional:
*- x-offset:
Untuk mengatur jarak bayangan secara horizontal (kanan kiri), ini nilainya nomerik. Kalo dikasih 10px, maka dia akan bergerak ke kanan sebanyak 10px. Sebaliknya kalo kasih -10px, maka dia akan bergerak ke kiri sebanyak 10px.
*- y-offset:
Untuk mengatur jarak bayangan secara vertikal (atas bawah), ini nilainya nomerik. Kalo dikasih 10px, maka dia akan bergerak ke bawah sebanyak 10px. Sebaliknya kalo kasih -10px, maka dia akan bergerak ke atas sebanyak 10px.
*- blur:
Untuk mengatur seberapa besar blur yang ingin kita kasih pada bayangannya. Semakin besar angkanya, semakin nge-blur.
*- color:
Untuk memberikan warna yang kita inginkan, bisa nama warna, Hex, maupun rgb()

Penjelasan opsional:
*- inset:
Digunakan untuk membuat bayangan berada didalam box modelnya
*- spread:
Untuk mengatur size (ukuran) bayangan

Multiple Shadow:
Kelebihan dari box-shadow bisa multiple shadow ialah memberikan lebih dari satu bayangan pada sebuah elemen
Multiple Border:
Ada juga multiple border, tampak seperti border padahal hanyala sebuah bayangan yang nilai blurnya kosong/nol (0)

Kode non-opsional:

selector { box-shadow: x-offset, y-offset, blur, color; } /* sesuaikan urutannya */
selector { box-shadow: 7px 7px 5px #000; } /* contohnya */

Output non-opsional:

Kode Opsional:

selector { box-shadow: inset, x-offset, y-offset, spread, blur, color; } /* sesuaikan urutannya */
selector { box-shadow: inset 0 0 35px 5px #00f; } /* contohnya */
selector { box-shadow: 10px 10px 5px #f00, -10px -10px 5px #00f; } /* contoh multiple */

*- Gradients

Digunakan untuk mengradasikan berbagai macam macam warna. Gradients terbagi menjadi 2 bagian, yaitu linear-gradient dan radial-gradient. Linear-gradient untuk memberikan gradasi warna secara garis lurus, sedangkan radial-gradient untuk memberikan gradasi warna secara melingkar. Nah kedua nilai ini diberikan pada property background-image, kenapa background-image bukannya background-color? supaya bisa menggunakan background-image dan background-color secara bersamaan

*- Linear-Gradient

Selengkapnya
1. Bisa menggunakan nilai dengan satuan derajat (deg)
2. Bisa menggunakan satuan arah, terdapat 4 bagian yaitu:
*- to top
*- to right
*- to bottom
*- to left
3. Bisa menggunakan arah diagonal, terdapat 4 bagian yaitu:
*- to left top
*- to right top
*- to left bottom
*- to right bottom
4. Bisa juga menggunakan color stop, untuk memberikan bobot warna yang berbeda beda, seperti output dibawah ini

Syntax-nya:

selector { background-image: linear-gradient(warna 1,warna 2,...,warna n); }

Kodenya:

selector { background-image: linear-gradient(#e74c3c, #9b59b6, #f1c40f, #d35400); }
selector { background-image: linear-gradient(to right, #4169e1, #d44); } /* satuan arah */
selector { background-image: linear-gradient(to right bottom, #00f, #90ee90); } /* arah diagonal */
selector { background-image: linear-gradient(to right, #87ceeb 63%, #32cd32 80%); } /* color stop */

Outputnya:

*- Radial-Gradient

Selengkapnya
Untuk tipe, jangkauan, posisi y serta posisi x ialah opsional
1. Penjelasan nilai tipe, terbagi 2 yaitu:
*- ellipse (default)
*- circle
2. Penjelasan jangkauan, jarak dari pusat gradient ke sisi terluar (sisi yang ga ada gradisanya), terbagi menjadi 4 yaitu:
*- closest-corner, jarak dari pusat gradient ke sudut terdekat elemen
*- closest-side, jarak dari pusat gradient ke sisi terdekat elemen
*- farthest-corner, jarak dari pusat gradient ke sudut terjauh elemen
*- farthest-side, jarak dari pusat gradient ke sisi terjauh elemen
3. Penjelasan posisi-y dan posisi-x:
*- posisi-y, terdapat nilai top, center, bottom dan satuannya pixel (px)
*- posisi-x, terdapat nilai left, center, right dan satuannya pixel (px)

Syntax lengkapnya:

selector { background-image: radial-gradient( [tipe] [jangkauan], at [posisi-y], [posisi-x], warna 1,warna 2, ..., warna n); }

Kodenya:

selector { background-image: radial-gradient(circle, #d35400 35%, #3498db 80%); } /* tipe */
selector { background-image: radial-gradient(circle closest-side, #f00 43%, #00f); } /* jangakauan */
selector { background-image: radial-gradient(circle farthest-side at top center, #080 33%, #800080 63%); } /* posisi y-x */
selector { background-image: radial-gradient(#d35400 20%, #3498db 37%, #2c3e50 80%); } /* color stop */

Outputnya:

*- Fonts

Digunakan untuk mengubah font pada teks menjadi berbeda, dan ada empat cara, seperti: menggunakan font biasa pada CSS, menggunakan google font, menggunakan @font-face serta menggunakan web font generator

Selengkapnya
Cara 1: Font Bawaan
Menggunakan font biasa atau bawaan hanya sedikit, kalian bisa tuliskan langsung di font-family fontnya seperti: arial, consolas, verdana dan juga georgia

Cara 2: Google Font
*- Klik website ini https://fonts.google.com/
*- Lalu pilih dan cari font yang ingin digunakan
*- Klik + select this style pada font yang dipilih
*- Klik menu 'Embed' dan pilih ingin menggunakan <link> atau @import
*- Disini saya contohkan menggunakan <link>, dan salin link rel-nya lalu paste didalam Tag <head>
*- Salin syntax di 'Embed' tadi dan pastekan di selector apa yang ingin berubah fontnya, misal: h1 { font-family: 'Chilanka', cursive; }
Notice: Menggunakan google font membutuhkan koneksi internet (online)

Cara 3: @font-face
*- Dengan menggunakan @font-face, membutuhkan file fontnya (download file font) dulu. Dengan begitu fontnya bisa digunakan secara offline
*- Download dan cari fontnya di https://www.fontsquirrel.com/
*- Pilih fontnya terlebih dahulu, dan klik Download OTF
*- Setelah di download, ekstrak filenya seperti biasa dan pindahkan ke folder ngoding kalian
*- Kemudian ketik syntax-nya di CSS, seperti ini: 
@font-face {
     font-family: "alex brush";
     src: url(font/AlexBrush-Regular.tff) format( 'truetype');
}
selector {
     font-family: "alex brush", arial, sans-serif;
}

Notice: file font saya terdapat di folder 'font', jadi buat dulu folder 'font'-nya atau hapus aja font/ pada url diatas

Apa bedanya format WOFF dengan TFF? Bedanya WOFF lebih cepat meng-load browsernya dibanding format TFF. WOFF artinya Web Open File Format, yang dikembangkan oleh mozilla khusus untuk website walaupun merupakan kompres dari tipe TFF/OTF

*- Filters

Filter adalah sebuah property untuk mengubah kondisi suatu objek atau gambar. Syntax-nya seperti ini: selector { filter: [tipe filter]; }

Selengkapnya
Beberapa tipe filter beserta satuannya:
*- blur, pixel (px)
*- brightness, (angka | %)
*- contrast, (angka | %)
*- saturate, (angka | %)
*- grayscale, (angka | %)
*- sepia, (angka | %)
*- hue-rotate, derajat (deg)
*- invert, (angka | %)
*- opacity, (angka | %)
*- drop-shadow, (sama seperti box-shadow nilainya)

Apa bedanya drop-shadow dengan box-shadow? drop-shadow bisa memberikan bayangan pada teks yang ada didalam image, sedangkan box-shadow hanya memberikan bayangan pada box image-nya aja bukan objek pada image-nya

*- Transition

Transisi adalah property yang memungkinkan kita dapat mengubah elemen ketika bergerak menjadi halus. Syntax-nya, seperti ini: selector { transition: [call] [duration] [function] [delay]. Dari ke-4 nilai tersebut ada yang wajib, yaitu durasi dan durasi memiliki 2 satuan, yaitu: second (s) dan milisecond (ms)

Selengkapnya
Penjelasannya:
Mendeklarasikan [call] ada 3, yaitu:
*- all, memanggil semua nama property
*- nama_property, memanggil salah satu nama property
*- none, tidak ada
Mendeklarasikan [function] ada 6, yaitu:
*- ease
*- ease-in
*- ease-out
*- ease-in-out
*- linear
*- cubic-bezier(w, x, y, z)
Mendeklarasikan [duration] dan [delay], yaitu:
*- second (s), untuk menghitung secara perdetik
*- milisecond (ms), untuk menghitung lebih rendah dari detik, contoh: 1000ms = 1 detik

Notice: Untuk fungsi cubic-bezier kalian bisa cek di 'inspeksi' atau CTRL + SHIFT + I dan atur nilainya (jika terdapat property transition pada suatu elemen)

Kodenya:

selector { transition: all 1s ease-in-out 500ms; }

*- Animation

Animasi sama halnya seperti transition, yang membedakan kedua property ini adalah keadaan awal dan akhirnya

Selengkapnya

Syntax-nya:

@keyframes identifier {
     from {
          property: value;
     }
     to {
          property: value;
     }

Syntax diatas ialah selector baru bukan property. Berikut nilai lengkap pada property animation, seperti:

Kodenya:

selector { animation: [identifier] [duration] [function] [delay] [iteration-count] [direction] [fill-mode] [play-state]; }
Berikut penjelasan nilai nilainya, kecuali duration, timing-function, dan delay (penjelasannya sama seperti di property transition)
*- identifier:
Identifier adalah sebuah nama buatan (bebas) yang harus sesuai dengan identifier pada @keyframes
*- iteration-count:
Untuk mengatur pengulangan kondisi animasi yang nilainya berupa "angka", mau berapa kali pengulangannya atau mengulang secara terus menerus dengan nilai infinite
*- direction:
*- normal, animasi akan berjalan dari awal ke akhir
*- reverse, animasi akan berjalan dari akhir ke awal
*- alternate, animasi akan berjalan dari awal ke akhir dan dari akhir ke awal
*- alternate-reverse, animasi akan berjalan dari akhir ke awal dan dari awal ke akhir
*- fill-mode:
Fill mode untuk menentukan akhir dari animasi seperti apa nantinya
*- none, tidak ada
*- forwards, berhenti di frame akhir
*- backwards
*- both
*- play-state:
*- running, animasi berjalan semestinya (default)
*- paused, animasi akan berhenti, untuk melakukan nilai ke-2 ini dibutuhkan bantuan javascript supaya bisa berhenti di tengah jalan animasinya

Kode Shorthand:

selector { animation: identifier 1s ease 2s infinite alternate forwards running; }

Kode property satuan:

     animation-name: identifier;
     animation-duration: 1s;
     animation-timing-function: ease;
     animation-delay: 2s;
     animation-iteration-count: infinite;
     animation-direction: alternate;
     animation-fill-mode: forwards;
     animation-play-state: running;
}

Responsive Layouts

*- Flexbox

Flexbox atau disebut juga fleksibel box layout modul adalah teknik CSS yang udah lama sejak 2016/2017. Flexbox membantu kita dalam mengatur posisi layout (box model) pada halaman website.

Definisi lainnya, flexbox adalah sebuah model layout 1 dimensi yang dapat mengatur jarak dan kesejajaran antar item dalam sebuah container. Apa yang dimaksud 1 dimensi? Yaitu hanya dapat mengatur 1 dimensi pada saat tertentu, diantaranya baris (horizontal) dan kolom (vertikal) tidak bisa sekaligus. Jadi, jika ingin mengatur baris maka baris aja, sebaliknya jika mengatur kolom maka kolom aja. Lalu bagaimana jika ingin mengatur ke-2 dimensi tersebut? Teknik itu menggunakan CSS Grid, karena CSS Grid dapat mengatur 2 dimensi.

Definisi lainnya menurut CSS Tricks adalah sebuah modul yang menawarkan cara yang lebih efektif untuk menyusun, mensejajarkan dan mendistribusikan jarak antar item dalam sebuah container, meskipun ukurannya dinamis atau bahkan kita ga tau

Selengkapnya
Istilah yang harus di perhatikan dalam flexbox, yaitu:
*- Wadah, wadah disebut juga sebagai container atau parent-nya
*- Items, elemen elemen yang didalam container, disebut juga sebagai child
*- Main Axis, mengatur elemennya secara horizontal (default)
*- Cross Axis, mengatur elemennya secara vertikal
*- Main Size, menentukan ukuran container-nya
*- Cross Size, menentukan ukuran kolomnya
*- Main Start dan Main End, menentukan awal dan akhir elemennya (secara horizontal)
*- Cross Start dan Cross End, menentukan awal dan akhir elemennya (secara vertikal)

Notice: tapi yang wajib diperhatikan ialah container (wadah) dan items-nya


Property untuk container-nya: Notice: simbol | artinya atau
*- display:
Display ialah property yang sudah kalian kenal, namun dengan flexbox nilai pada display yaitu flex atau inline-flex. Ketika wadahnya ber-display: flex; maka akan membuat elemen didalamnya berperilaku flex juga
*- flex-direction:
Untuk mengatur arah atau urutan elemen didalam wadahnya, bisa secara horizontal atau kebalikkannya, dan vertikal atau kebalikkannya.
Nilainya: { flex-direction: row | row-reverse | column | column-reverse; }
*- flex-wrap:
Memungkinkan untuk memindahkan items ke baris dibawahnya atau kolom disampingnya, ketika elemennya sudah tidak muat lagi pada wadahnya. jika tidak di menambahkan property ini, maka elemennya akan memaksa terus di baris/kolom tersebut walaupun wadahnya udah ga muat lagi.
Nilainya: { flex-wrap: nowrap | wrap | wrap-reverse; }
*- justify-content:
Untuk mengatur jarak antar elemen elemennya terhadap main axis. Nilainya:
*- flex-start, rata kiri (default)
*- flex-end, rata kanan
*- center, rata tengah
*- space-between, elemen pertama mepet kiri, elemen terakhir mepet kanan dan sisanya diberi jarak yang sama
*- space-around, membagi setengah jarak sisi kiri dan kanannya
*- space-evenly, membagi sama rata jaraknya sisi kiri dan kanan
*- align-items:
Untuk mengatur kesejajaran items terhadap cross axis. Nilainya:
*- flex-start, rata ke atas (default)
*- flex-end, rata ke bawah
*- center, rata tengah
*- stretch, memenuhi wadanya ketika tidak diberi property height
*- baseline, untuk mengatur rata tengah berdasarkan teksnya
*- align-content:
Untuk mengatur jarak antar items terhadap cross axis, berfungsi ketika itemsnya lebih dari satu baris/kolom.
Nilainya: { align-content: flex-start | flex-end | center | stretch | space-between | space-around; }

Property untuk itemsnya:
*- order:
Untuk mengatur urutan itemsnya. Nilainya angka
*- flex atau flex-grow:
Untuk mengatur ukuran dari itemsnya, dan flex adalah shorthandnya
*- align-self:
Untuk mengatur satu elemen yang ingin disejajarkan pada sebuah items yang spesifik terhadap cross axis. Nilainya:
*- flex-start, rata ke atas (default)
*- flex-end, rata ke bawah
*- center, rata tengah
*- stretch, memenuhi ukuran width dan height-nya

Kodenya:

.container {
     display: flex;
     justify-content: space-around;
     align-items: center;
     flex-direction: column;
     flex-wrap: wrap;
}

*- CSS Grid

CSS grid adalah modul CSS baru, untuk mendefinisikan sistem layout berbentuk grid 2 dimensi

Selengkapnya
Istilah dan terminologi dalam CSS grid:
*- Grid Container:
Sebuah elemen pembungkus grid yang didefinisikan dengan menuliskan property display bernilai grid
*- Grid Items:
Sebuah elemen yang berada (1 level) didalam grid container
*- Grid Line:
Sebuah garis horizontal atau vertikal yang memisahkan grid menjadi beberapa bagian dan ditandai dengan angka
*- Grid Cell:
Potongan atau pertemuan antara baris dan kolom didalam grid
*- Grid Area:
Sebuah kumpulan yang lebih dari satu grid cell yang membentuk kotak
*- Grid Track:
Sebuah ukuran atau jarak antara grid line, entah itu horizontal atau vertikal
*- Grid Gap:
Jarak antara grid track atau grid cell

Penjelasan dan fungsi grid pada container:

Selengkapnya

Property untuk kolom dan baris:

*- grid-template-columns dan grid-template-rows:
Kedua property itu mendefinisikan kolom/baris ngan cara menuliskan nilainya yang dipisah oleh spasi. Nilai tersebut merepresentasikan ukuran grid dan spasi merepresentasikan grid line. Nilainya ada 2, yaitu:
*- track-size, berupa satuan px, %, auto, fr (fraction)
*- line-name, nama yang bisa diberikan pada track

Perbedaan Explicit Grid dengan Implicit Grid adalah explicit secara jelas memberikan ukuran dari gridnya, sedangkan implicit gridnya menentukan ukurannya secara auto atau otomatis

Property auto kolom dan baris:

*- grid-auto-columns dan grid-auto-rows:
Kedua property itu untuk mengatur ukuran grid track yang tidak dituliskan pada grid template-nya atau disebut implicit track

Property auto flow:

*- grid-auto-flow:
Untuk mengatur penempatan item atau cell pada grid track, termasuk yang ditulis secara implicit.
Nilainnya: { grid-auto-flow: row | columns | row dense | column dense; }

Property area dan gap:

*- grid-template-areas:
Mendefinisikan grid template menggunakan nama dari area yang dituliskan pada property grid-area didalam itemnya
*- grid-template:
Sebuah shorthand untuk mendefinisikan grid-template-rows, grid-template-columns, dan grid-template-areas dalam satu deklarasi.
Nilainya: { grid-template: grid-template-rows | grid-template-areas / grid-template-columns; }
*- grid-column-gap, grid-row-gap, column-gap dan row-gap:
Mendefinisikan ukuran grid line atau jarak antara baris dan kolom
*- grid-gap atau gap:
Adalah sebuah shorthand untuk mendefinisikan column-gap dan row-gap dalam satu deklarasi
*- grid:
Shorthand yang mendefinisikan semua property grid dalam satu deklarasi.
Nilainya: { grid: grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns, grid-auto-flow; }

Property grid alignment: (Property ini mirip dengan perilaku dari flexbox)

*- justify-items:
Mensejajarkan grid-items pada sumbu horizontal. Nilainya:
*- start, mepet kiri
*- end, mepet kanan
*- center, berada di tengah secara horizontal
*- stretch, nilai ini tidak berlaku jika diberikan lebar (width)
*- align-items:
Mensejajarkan grid-items pada sumbu vertikal.
Nilainya: { align-items: start | end | center | stratch
Penjelasannya sama seperti justify-items
*- place-items:
Shorthand untuk menuliskan justify-items dan align-items dalam satu deklarasi
*- justify-content:
Untuk mengatur posisi seluruh grid container pada sumbu horizontal dan dilakukan ketika ukuran total grid lebih kecil dari ukuran containernya, biasanya etika grid-items-nya menggunakan ukuran yang fixed. Nilainya:
*- start
*- end
*- center
*- stretch
*- space-around, memberikan setengah jarak untuk tiap tiap ujung kolomnya
*- space-between, kiri kanannya mepet dan sisanya diberi jarak (tengah)
*- space-evenly, sama rata jaraknya
*- align-content:
Penjelasannya sama seperti justify-content, tapi align-content secara vertikal. Nilainya: { align-content: start | end | center | stretch | space-around | space-between | space-evenly; }
*- place-content:
Shorthand untuk menuliskan justify-content dan align-content dalam satu deklarasi (sama seperti place-items)

Penjelasan dan fungsi grid pada items:

Selengkapnya
*- grid-column-start & grid-column-end | grid-row-start & grid-row-end:
Untuk menentukan tempat dari grid item dengan mengacu pada nomor atau nama grid line yang spesifik. Nilainya:
*- grid-column-start: nomor | nama | span nomor | span nama | auto;
*- grid-column-end: nomor | nama | span nomor | span nama | auto;
*- grid-row-start: nomor | nama | span nomor | span nama | auto;
*- grid-row-end: nomor | nama | span nomor | span nama | auto;

Penjelasan nilainya:
*- nomor | nama, menentukan nomor/nama yang diberikan pada grid line
*- span nomor, item akan memanjang atau melebar sampai sejumlah nomornya
*- span nama, item akan memanjang atau melebar sampai nama grid line tertentu
*- auto, implicit grid

*- grid-column dan grid-row:
Shorthand untuk start end pada grid-column dan grid-row dalam satu deklarasi. Nilainya:
*- grid-column: start-line / end-line | start-line / span value;
*- grid-row: start-line / end-line | start-line / span value;

*- grid-area:
Menentukan nama area pada item sesuai dengan template yang sudah dibuat sebelumnya melalui property grid-template-areas dan ini juga bisa digunakan sebagai shorthand dari grid-column-start & grid-column-end | grid-row-start & grid-row-end. Nilainya:
*- grid-area: nama_template;
*- grid-area: row-start / column-start / row-end / column-end;

Contohnya:
*- grid-row-start: 2;
*- grid-column-start: 1;
*- grid-row-end: 4;
*- grid-column-end: 3;

Shorthand:
*- grid-area: 2 / 1 / 4 / 3;

*- justify-self:
Untuk mengatur posisi item pada sebuah cell terhadap sumbu horizontal. Nilainya:
*- start, berada diawal (sebelah kiri) itemnya
*- end, berada diakhir (sebelah kanan) itemnya
*- center, berada di tengah itemnya
*- stretch, memenuhi itemnya dan stretch nilai default

*- align-self:
Untuk mengatur posisi item pada sebuah cell terhadap sumbu vertikal dan penjelasan nilainya sama seperti justify-self (align-self secara vertikal atas bawah). Nilainya: { align-self: start | end | center | stretch; }

*- place-self:
Shorthand untuk justify-self dan align-self dalam satu deklarasi. Nilainya: { place-self: align-self justify-self }

Contohnya:
*- place-self: start center;
*- place-self: stretch;
Jika kondisi nilainya berbeda, maka tuliskan kedua nilai tersebut. Sedangkan kondisi nilainya sama, maka cukup panggil satu nilai aja (walaupun ditulis keduanya ga masalah)

Penjelasan spesial function pada CSS grid:

Selengkapnya
*- repeat():
Untuk menentukan ukuran grid track secara berulang. Contohnya:
Tanpa function repeat():
*- grid-template-columns: 1fr 1fr 1fr 1fr 1fr;

Menggunakan function repeat():
*- grid-template-columns: repeat(5, 1fr); /* atau */
*- grid-template-columns: repeat(2, 1fr), repeat(3, 1.5fr); /* boleh beda ukuran */

*- min-content dan max-content:
Untuk menentukan seberapa besar ukuran pada grid tracknya berdasarkan content pada sebuah item. Nilainya: min-content | max-content

*- auto-fill dan auto-fit:
Untuk menentukan jumlah item untuk berada ada grid track dan biasanya digunakan didalam nilai function repeat(). Contohnya:
*- grid-template-columns: repeat(auto-fill, 2fr);
*- grid-template-columns: repeat(auto-fit, 2fr);

*- minmax():
Untuk menentukan ukuran minimal dan maximal dari grid track. Contohnya
*- grid-template-columns: 1fr minmax(200px, 300px) 1fr;

*- Media Queries

Media query adalah sebuah selector yang bertugas untuk membuat website responsive. Media query mempunyai 2 property, yaitu: min-width dan max-width

Selengkapnya
*- Penjelasan property min-width:
Jika ukuran lebarnya lebih kecil dari 600px, maka pake codingan diluar @media. Sedangkan ukuran lebarnya lebih besar dari 600px, maka pake codingan yang didalam @media
*- Penjelasan property max-width:
Jika ukuran lebarnya lebih kecil dari 600px, maka pake codingan yang didalam @media. Sedangkan ukuran lebarnya lebih besar dari 600px, maka pake codingan yang diluar @media

Notice: Biasanya ukurannya tergantung apa yang ingin di responsive-kan, contoh: website di laptop reponsive ke mobile, atau website mobile responsive ke tablet, dan semacamnya

Kodenya:

selector {
     /* codingan diluar @media */
}
@media screen and (max-width: 600px) {
     selector {
          /* codingan didalam @media */
     }
}

*- CSS Units

CSS Units atau CSS Value adalah sebuah angka nilai untuk memastikan ukuran elemennya. Nilai yang bisa diberikan, seperti: integer, number, percentage dan dimension

Selengkapnya
Penjelasannya:
1. Integer:
Bilangan bulat (positif/negatif) serta hanya menuliskan nilainya aja, contoh:
*- z-index: -10;
*- font-weight: 600;
*- order: 3;

2. Number:
Hanya angka aja tapi pecahan, contoh:
*- opacity: 0.7;
*- transform: scale(2.5);
*- transition: 0.5s;

3. Percentage:
Merepresentasikan sebagian dari nilai tertentu. Persen selalu relatif terhadap nilai yang lain, yaitu pembungkusnya (parent/wadah), contoh:
*- width: 100%;
*- transform: translate(-50%, -50%);

4. Dimension:
Sebuah angka (number) yang memiliki satuan dibelakangnya, dan ada 4 kategori untuk dimension, yaitu:
*- Angle
*- Time
*- Resolution
*- Lenght
Penjelasan dari dimension:
a). Angle (sudut):
Salah satu satuannya yaitu derajat (deg), radiant (rad), gradian (grad) dan turn. Contohnya:
*- transform: rotate(45deg);
*- transform: rotate(0.523333333rad);
*- transform: rotate(33.3grad);
*- transform: rotate(0.08333333turn);
b). Time (waktu):
Satuannya adalah second (s) dan ms (milisecond). Contohnya:
*- transition: 500ms;
*- transition: 1s;
c). Resolution (resolusi):
Menentukan resolusi dari tampilan website. Satuannya dpi (dot per inch), dpcm (dot per centimeter) dan dppx (dot per pixel). Contohnya:
*- @media print and (min-resolution: 300dpi) { ... }
d). Lenght:
Terdapat 2 cabang lagi, yaitu nilai absolute dan nilai relative.
*- Lenght - Nilai Absolute:
*- px (pixel)
*- pt (point)
*- pc (pica)
*- cm (centimeter)
*- mm (milimeter)
*- in (inches)
*- Lenght - Nilai Relative:
*- %: persentase ukurannya relatif terhadap ukuran pembungkusnya
*- em: relatif terhadap ukuran font yang sedang digunakan
*- rem: relatif terhadap ukuran font pada <html> atau CSS reset (:root)
*- ch: relatif terhadap lebar karakter '0' dari font yang sedang digunakan
*- vh: relatif terhadap tinggi (height) dari viewport
*- vw: relatif terhadap lebar (width) dari viewport
*- vmin: relatif terhadap dimensi terkecil sebuah viewport
*- vmax: relatif terhadap dimensi terbesar sebuah viewport

*- CSS Modern

CSS modern trik adalah trik khusus web desainer. Terdapat beberapa property CSS baru dan syaratnya ialah harus menggunakan browser versi terbaru (apapun itu browsernya), karena untuk CSS modern ini bisa dibilang baru dan mungkin aja belom support ke semua browser. Oleh karena itu, mungkin dari beberapa property tersebut ada yang memerlukan vendor prefixes

Selengkapnya
1. Clipping Path:
Untuk menyembunyikan bagian dari elemen menggunakan path dan nama property-nya adalah clip-path. Nilainya: { clip-path: inset() | circle() | ellipse() | polygon(); }

Penjelasan nilainya:
*- inset(): parameternya sama seperti box model, yaitu 4 sisi.
Contoh: clip-path: inset(100px 0 50px 0);

*- circle(): parameternya satu aja.
Contoh: clip-path: circle(200px);

*- ellipse(): parameternya 2 (x, y), nilai pertama horizontal dan nilai kedua vertikal.
Contoh: clip-path: ellipse(200px, 400px);

*- polygon(): parameternya tak terhingga tapi minimal 3 (x y, x y, x y).
Contoh: clip-path: polygon(50% 0%, 0% 100%, 100% 100%);
Notice: Untuk polygon() ini terdapat website untuk membantu memperhitungkan nilainya, saya sudah sediakan di menu help

Kodenya:

selector { clip-path: polygon(11% 13%, 66% 4%, 91% 16%, 94% 79%, 71% 59%, 50% 91%, 46% 57%, 10% 68%, 35% 36%); }

Outputnya:

2. Clipping Text:
Untuk menyembunyikan bagian tertentu dari elemen menggunakan property background-clip dan nilainya text. background-clip dibantu dengan property satu lagi, yaitu text-fill-color bernilai transparent. Clipping Text ini membutuhkan vendor prefix, karena ada browser yang belom support property ini

Kodenya:

selector {
     -webkit-background-clip: text;
     -webkit-text-fill-color: transparent;
}

Outputnya:

Hello World!

3. Text Stroke:
Membuat teks mempunyai sebuah border. Property-nya bernama text-stroke dengan nilai angka (dimensionnya) serta color

Kodenya:

selector { -webkit-text-stroke: 2px #0f0; }

Outputnya:

Hello World!

4. Masking Image:
Dengan menggunakan property ini, kalian membutuhkan gambar dengan extension .svg dan lebih baik lagi jika gambarnya hanya satu warna aja. Nama property-nya mask
Contoh: { mask: url(image.svg); }

lalu bisa ditambah property mask-repeat: no-repeat dan masih banyak lagi property tambahannya
5. Shapping:
Membuat tulisannya atau teksnya mengelilingi gambarnya, tapi tidak menganggap gambarnya punya areanya. Property-nya ada 2, yaitu shape-outside dan shape-inside. Lalu nilainya sama dengan clip-path
